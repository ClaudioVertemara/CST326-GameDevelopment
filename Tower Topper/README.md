Build a tower with blocks that fall from the sky and make the tallest tower you can without it leaning over.          

# Controls
Android:    
Touch Screen - Navigate menu    
UI Arrows - Move blocks left and right

PC:    
"A" - Move block left    
"D" - Move block right    
Mouse - Navigate menu
